# FinFlow 💸

**FinFlow** is a personal finance management web application that helps users track their income and expenses, analyze spending patterns, and manage their financial records efficiently.

Built with **React.js** and **Firebase**, FinFlow provides a smooth and secure way to monitor financial transactions in real time.

---

## 🚀 Features

* 🔐 **Authentication**

  * Email & Password login/signup using Firebase Authentication
  * Google Sign-in support
  * Secure user authentication system

* 💰 **Transaction Management**

  * Add income and expense transactions
  * Delete transactions
  * Categorize transactions for better tracking

* 📊 **Dashboard Overview**

  * View total balance
  * Total income and expenses summary
  * Quick financial insights

* 📋 **Transaction History**

  * Display transactions in a structured table using Ant Design
  * Sort and view transaction details easily

* 🪟 **Interactive UI**

  * Modal forms for adding transactions
  * Clean and responsive interface using Ant Design components

* ☁️ **Cloud Storage**

  * Store and manage transactions in Firebase Firestore
  * Real-time database updates

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/iamrohitm/finflow.git
```

Go to project folder:

```bash
cd finflow
```

Install dependencies:

```bash
npm install
```

Create `.env` file:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

Run the project:

```bash
npm run dev
```
